# Frontend Internship test task - Weather App

## Run Locally

Clone the project

```bash
  git clone https://github.com/masiuk-mykola/Weather-app.git
```

Go to the project directory

```bash
  cd Weather-app
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```

To view the app, visit http://localhost:1234/
